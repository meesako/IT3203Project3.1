About the Author

This site was created by Kaseem Rogers as part of the IT 3203 course project. The goal is to explain core web development ideas in simple terms and practice building a multi-page website with HTML, CSS, and JavaScript.

While working on this project, I focused on:

  • Building a consistent navigation menu across all pages
  • Making the layout responsive so it works on phones and laptops
  • Using a single CSS file to keep the design organized
  • Writing JavaScript to make an interactive quiz
