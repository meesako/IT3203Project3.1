HTML and CSS Basics

HTML (HyperText Markup Language) gives structure to a page. It defines headings, paragraphs, lists, links, images, and more. CSS (Cascading Style Sheets) controls the visual presentation, including colors, fonts, spacing, and layout.

HTML: Structuring the Page

Some common HTML elements include:

  • <h1>...</h1> – headings
  • <p>...</p> – paragraphs of text
  • <a href="...">...</a> – links
  • <img src="..." alt="..." /> – images
  • <ul> <li>...</li> </ul> – lists

CSS: Styling the Page

CSS separates design from content. A single CSS file can style many pages, which makes it easier to keep a site consistent and easier to maintain.

Common CSS properties include:

  • color, background-color
  • font-family, font-size
  • margin, padding
  • display, flex, grid

In this project, all pages share the same styles.css file so the navigation and layout look consistent, which was one of the project requirements.
